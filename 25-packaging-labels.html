25.1 Purpose

Packaging represents the brand when the customer receives the product. It is the first physical touchpoint and must communicate quality, precision, and trust. All packaging must follow the established Koleex brand systems.

Brand Systems Applied to Packaging

Logo System — Correct placement, clear space, and minimum size on all surfaces
Color System — Brand palette applied consistently across all packaging types
Typography System — Approved fonts with correct hierarchy and weights
Layout & Grid — Structured grid-based layouts with proper margins
Graphic Elements — Approved patterns, lines, and textures only
Photography — High-quality, on-brand product photography
Icon System — Consistent icon style for specifications and features

Clear hierarchy with consistent size steps

25.9 Do & Don't

Follow these rules strictly to maintain brand consistency across all packaging and labeling materials.

Do
  • Use approved brand fonts on all packaging
  • Follow the logo clear space and minimum size rules
  • Maintain grid-based, structured layouts
  • Use black/white/gray as primary packaging colors
  • Include all required certification marks
  • Use high-quality product photography
  • Apply consistent typography hierarchy
  • Include barcode, QR code, and serial number
  • Use approved handling and warning symbols
  • Maintain large margins and breathing room
  • Scale design proportionally for different box sizes
  • Use minimal color accents sparingly
  • Print test samples before mass production
  • Ensure text is legible at all print sizes
Don't
  • Use non-brand fonts or decorative typefaces
  • Stretch, rotate, or distort the logo
  • Use bright, neon, or cartoon colors
  • Create cluttered or overcrowded layouts
  • Place elements without grid alignment
  • Use low-resolution or pixelated images
  • Mix multiple font families on one surface
  • Omit required legal or certification information
  • Use generic or non-brand stock photography
  • Apply drop shadows or 3D effects to text
  • Use handwritten or informal labels
  • Ignore minimum text size for readability
  • Use inconsistent spacing or margins
  • Add decorative borders or unnecessary graphics
