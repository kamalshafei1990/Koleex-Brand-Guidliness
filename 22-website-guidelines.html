Website Design Guidelines

Guidelines for designing and maintaining the Koleex corporate website with consistent brand identity across all pages and devices.

Overview

The website is one of the primary digital platforms representing the Koleex brand. It is often the first place where customers, partners, and visitors interact with the company. Therefore, the website must fully reflect the brand identity, visual system, tone, and structure defined in the brand guidelines.

The website should communicate the brand as:

Modern Technological Structured Global Professional Innovative

Design Principles

Minimal design
Structured layout
Grid system
Clean typography
Large spacing
Clear hierarchy
Limited colors
Strong visual consistency
Professional UI style
Product-focused presentation

Important

The website must not look like a template website. It must look like a technology company website.

Design Principles

All Koleex website pages must follow these principles:

Do
  • Clean layout
  • Large white space
  • Grid-based design
  • Clear typography hierarchy
  • Minimal colors
  • Consistent margins
  • Consistent spacing
  • Structured content blocks
  • Modular layout system
  • Strong visual hierarchy
  • Clear navigation
  • Easy readability
  • Product-focused visuals
  • Professional photography
  • Consistent icon style
  • Consistent UI elements
  • Responsive design
  • Fast loading pages
  • Simple user experience
  • Logical page structure
Avoid
  • Cluttered pages
  • Too many colors
  • Too many animations
  • Decorative graphics
  • Template-style layouts
  • Stock photo heavy layouts
  • Random layout styles
  • Inconsistent spacing
  • Inconsistent typography
  • Inconsistent button styles
Page Layout Structure

Typical page structure follows a consistent flow from top to bottom. Each section should have clear spacing, grid alignment, balanced layout, clean typography, consistent margins, and structured content blocks.

The header should include: Logo, Main navigation menu, Language selector, Contact button or call to action, Optional search icon, and Optional login button. Do / Don't Rules

Summary of all website design rules to ensure consistency and quality across all website pages and components.

Do
  • Use clean, minimal layout
  • Follow grid-based design system
  • Maintain large white space
  • Use clear typography hierarchy
  • Keep colors minimal (black, white, silver)
  • Ensure consistent margins and spacing
  • Use structured content blocks
  • Maintain strong visual hierarchy
  • Use product-focused professional photography
  • Keep consistent icon and UI element styles
  • Design for all screen sizes (responsive)
  • Optimize for fast loading
  • Keep simple user experience
  • Use logical page structure
  • Include clear navigation
  • Use modular layout system
Don't
  • Create cluttered or crowded pages
  • Use too many colors
  • Add excessive animations
  • Include decorative graphics
  • Use template-style layouts
  • Rely heavily on stock photos
  • Apply random layout styles
  • Use inconsistent spacing
  • Mix typography styles
  • Use inconsistent button styles
  • Use low quality images
  • Add cartoon graphics or illustrations
  • Use bright colorful photography
  • Over-edit or over-process images
  • Ignore responsive design
  • Create complex user flows
