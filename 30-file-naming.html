File Naming & Structure

A comprehensive file naming and folder structure system for all brand files — ensuring consistency, organization, and professional workflow across every team and department.

30.1 — Purpose

The KOLEEX file naming and folder structure system ensures that every brand asset — from logos to technical documents — is organized, searchable, and consistently named across all teams, departments, and external partners.

One system. Every team. Every file.

This system is mandatory for designers, marketing teams, engineers, management, printing companies, external agencies, and all partners working with KOLEEX brand files.

Designers Marketing Engineers Management Printing Companies External Partners

30.2 — General Rules

All KOLEEX brand files must follow these mandatory naming rules without exception. Consistency in naming prevents confusion, saves time, and maintains professionalism.

English only — all file names must be in English
Brand name always written as capital KOLEEX
Use hyphen “-” to separate words in the file name
Use underscore “_” only for version numbers
No spaces allowed in any file name
No special characters (!@#$%^&*) in file names
No Chinese, Arabic, or non-English characters
No random, personal, or abbreviated names
Correct Naming
KOLEEX-Brochure-Industrial-Machines-V1.pdf
KOLEEX-Logo-Primary-Black-V2.svg
KOLEEX-Catalog-Product-2026-V1.pdf
KOLEEX-Poster-Exhibition-Shanghai-V3.ai
Wrong Naming
brochure final FINAL (2).pdf
logo_new_v2_FINAL_USE THIS.svg
catalog ahmed edit march.pdf
poster copy - Copy (3).ai

Never use these in file names

The following naming patterns are strictly prohibited across all departments and partners.

final new last version copy edit draft test use this one for print updated

Every KOLEEX file must follow this standard naming formula. The structure ensures that any person on any team can immediately understand the content, category, and version of any file.

30.5 — Version Control

All KOLEEX files use a simple, clear version control system. Versions are numbered sequentially: V1, V2, V3, and so on. Never use words like “final”, “new”, or “last version” in any file name.

V1
Initial Release
V2
First Revision
V3
Second Revision
V4
Current

Never use these version labels

Words like “final”, “new”, “latest”, “last version”, “updated”, or “revised” are strictly prohibited. Only numeric versions (V1, V2, V3...) are allowed.

Correct Versions
KOLEEX-Brochure-Products-V1.pdf
KOLEEX-Brochure-Products-V2.pdf
KOLEEX-Brochure-Products-V3.pdf

KOLEEX-Brochure-Products-FINAL.pdf

KOLEEX-Brochure-Products-last-version.pdf

30.9 — Do & Don't

Follow these rules strictly when naming and organizing all KOLEEX brand files. Violations create confusion and damage brand professionalism.

Do
  • Always start file names with KOLEEX
  • Use English only for all file names
  • Use hyphens to separate words
  • Use the official category names
  • Use V1, V2, V3 for version control
  • Include the year in the file name
  • Store files in the correct numbered folder
  • Use approved file formats only
  • Keep file names descriptive and clear
  • Update version numbers with each revision
  • Archive old versions in a sub-folder
  • Follow the standard naming formula exactly
Don't
  • Never use spaces in file names
  • Never use special characters (!@#$%)
  • Never use Chinese or Arabic in file names
  • Never use personal names in file names
  • Never use “final” or “new” as versions
  • Never create random folder structures
  • Never rename official brand asset files
  • Never store brand files on personal devices only
  • Never use abbreviated or unclear names
  • Never skip version numbers
  • Never mix categories in one folder
  • Never use lowercase for the brand name

File Naming & Structure Summary

English Only Hyphen Separated V1, V2, V3 33 Categories Master Assets No Spaces

Every file produced for KOLEEX — whether internal or external — must follow this naming and structure system. Consistency in file management reflects the same professionalism and precision that defines the KOLEEX brand.
