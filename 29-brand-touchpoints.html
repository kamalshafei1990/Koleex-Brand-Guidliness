Brand Touchpoints

Every interaction point where people experience the Koleex brand — from digital platforms to physical environments.

What Are Brand Touchpoints?

Brand Touchpoints are every point where a person, customer, partner, employee, or visitor interacts with the Koleex brand. A touchpoint is any moment where someone sees, reads, uses, visits, or receives communication from the brand.

Sees the brand
Reads the brand
Uses the brand
Visits the brand
Receives communication
Uses a product
Attends an exhibition
Visits the website
Opens an email
Receives a package
Enters the office
Uses the software
Sees a vehicle
Sees an advertisement
Receives a document

Every touchpoint must look, feel, and communicate the same brand identity.

Brand consistency is not only about logo and colors. It is about the entire experience.

28.3 Brand Application Principles

All brand applications must follow these core principles to maintain a unified, professional identity across every touchpoint.

Consistency — Same visual identity across all applications
Simplicity — Clean, uncluttered designs with clear messaging
Minimal — Use only essential elements, avoid overdesign
Grid Alignment — All elements placed on the brand grid system
Correct Logo — Use approved logo versions with proper clear space
Correct Color — Use only brand-approved color palette
Correct Typography — Use brand font with proper hierarchy
High-Quality Printing — Ensure premium print production
Professional — Every application reflects industrial precision
Industrial/Tech Feeling — Maintain the brand's technical character
Consistent Placement — Logo and elements in approved positions
Maintain Spacing — Respect clear zones and margins
Use Templates — Follow approved brand templates for all materials

28.6 Typography Usage

Typography across all brand applications follows the Koleex brand font system with clear hierarchy and consistent usage rules.

BOLD 700 Titles & Headlines
MEDIUM 500 Subtitles & Labels
REGULAR 400 Body text, descriptions, and supporting content
Use brand font only — Inter for all applications
Maintain clear hierarchy: titles, subtitles, body
Consistent spacing and line heights across materials
Left-aligned text as default (RTL for Arabic)
Avoid too many font sizes on one material
Never use decorative or script fonts
Bold for titles and headings only
Regular weight for body text and descriptions
28.7 Do & Don't

Follow these rules strictly to maintain brand consistency across all brand application materials.

Do
  • Use approved brand fonts on all materials
  • Follow the logo clear space and minimum size rules
  • Maintain grid-based, structured layouts
  • Use black/white/gray as primary brand colors
  • Keep designs minimal and professional
  • Use high-quality photography and imagery
  • Apply consistent typography hierarchy
  • Follow brand templates for all materials
  • Maintain industrial/tech aesthetic
  • Use high-quality printing and production
  • Scale design proportionally for different sizes
  • Maintain large margins and breathing room
  • Test materials before mass production
  • Keep consistent placement of elements
Don't
  • Use non-brand fonts or decorative typefaces
  • Stretch, rotate, or distort the logo
  • Use bright, neon, or cartoon colors
  • Create cluttered or overcrowded layouts
  • Place elements without grid alignment
  • Use low-resolution or pixelated images
  • Mix multiple font families on one surface
  • Ignore brand templates and guidelines
  • Use generic or non-brand stock photography
  • Apply drop shadows or 3D effects to text
  • Use inconsistent spacing or margins
  • Add decorative borders or unnecessary graphics
  • Use different color palettes per department
  • Create materials without brand approval
