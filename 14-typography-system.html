Typography System

The official typography system defining how all text appears across the Koleex brand — from posters and websites to software interfaces, documents, packaging, and signage.

TYPOGRAPHY IDENTITY SYSTEM

Typography is one of the strongest visual identity elements of the Koleex brand.

Typography defines how all text appears across the brand including posters, websites, software interfaces, documents, packaging, presentations, signage, and advertisements. Every typographic decision must communicate professionalism, technology, structure, clarity, precision, global identity, engineering mindset, minimal design, and premium industrial style.

Professionalism Technology Structure Clarity Precision Global Identity Engineering Mindset Minimal Design Premium Industrial
Primary Font: Helvetica Neue

Helvetica Neue is the primary and main font of the Koleex brand. It must be used across all brand materials and touchpoints. For digital applications where Helvetica Neue is unavailable, Inter serves as the approved fallback font.

Typography Style Rules

All Koleex typography must follow these structural rules to maintain a consistent, professional appearance. Typography should look like Apple, Sony, BMW, and IBM style typography systems.

Typography Should Be
  • Clean
  • Minimal
  • Structured
  • Well spaced
  • Left aligned in most cases
  • Grid aligned
  • Consistent spacing
  • Consistent line height
  • Consistent letter spacing
  • Clear hierarchy
  • Easy to read
  • Professional
  • Structured typography system
Typography Should Not Be
  • Decorative
  • Stylized
  • Handwritten
  • Condensed
  • Expanded
  • Italic heavy
  • Too many font weights
  • Too many font sizes
  • Unstructured
  • Randomly sized
  • Cluttered
  • Trendy or gimmicky

Reference Standard

Koleex typography should look like the typography systems used by Apple, Sony, BMW, and IBM — clean, minimal, structured, and built for clarity at every scale.

Typography Do

Always follow these typography rules across all Koleex brand materials:

Always use Helvetica Neue as the primary font across all materials
Use Inter as the approved digital fallback when Helvetica Neue is unavailable
Maintain a clear typographic hierarchy: Title, Subtitle, Body, Caption
Keep text left-aligned in most contexts for optimal readability
Use consistent line height across all text levels (1.5 to 1.8 for body text)
Use consistent letter spacing — tight for headlines, normal for body text
Align text to the grid system for structured, predictable layouts
Use bold weight (700) for titles and headings only
Use medium weight (500) for subtitles, labels, and navigation
Use regular weight (400) for body text and paragraphs
Use light weight (300) for captions, notes, and secondary information
Ensure all text meets minimum contrast requirements for accessibility
Use the defined font sizes from the typography hierarchy consistently
Keep typography clean, minimal, and structured at all times
Test typography at multiple sizes and on multiple devices before finalizing
Typography Don't

Never do any of the following in Koleex brand materials:

Do not use decorative, handwritten, or script fonts for any brand material
Do not use more than two font weights in a single layout
Do not use condensed or expanded font variants
Do not use italic as a primary text style — use sparingly for emphasis only
Do not mix multiple typefaces in the same design
Do not use text shadows, glows, outlines, or gradient text effects
Do not use text smaller than 12px for any application
Do not use centered alignment for long paragraphs — left-align body text
Do not use all caps for body text — reserve uppercase for labels and headings
Do not use inconsistent spacing between text elements
Do not place text over busy or complex backgrounds without sufficient contrast
Do not stretch, skew, or distort text in any way
Do not use colored text on colored backgrounds (maintain black/white/gray text)
Do not introduce new fonts without written approval from the brand team
